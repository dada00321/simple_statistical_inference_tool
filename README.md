# 推論統計小幫手

### 功能:
1) 隨機集區設計: 幫助列出〈隨機集區設計〉的 ANOVA Table
2) （卡方）配適度檢定: 卡方檢定 - 配適度檢定 求出 [檢定統計量], [預期的卡方值]，並 [計算實際的卡方值], [做出結論]
3) 簡單線性迴歸:  求出 [檢定統計量], [預期的卡方值]，並 [求出標準誤], [做出結論]
再進一步 [求出預測平均數 (Y_bar) 的信賴區間], [求出個別觀測值 (Y_i) 的預測區間]

### 補充:
本學期的統計課, 還有教到 卡方檢定 的 〈獨立性檢定〉,〈同質性檢定〉

目前還沒寫這兩個問題的計算小工具, 有機會再補上
